# TO-DO
RESTful To-Do List API project developed with Golang

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   go mod download
   ```
3. Run the application:
   ```bash
   go run main.go
   ```
4. The server will start on port 5000
